  rem = default bootstrap measurement

  bootstrap ignore smaller margins, so avoid margin-top

  box-sizing: border-box .. as default



spacing

  padding
    p{dir}-{amt}
      dir=direction
        t=top
        e/r=end/right
        b=bottom
        s/l=start/left
        x = each side horizontally
        y = each side vertically
      amt=amount (0-5)
  margin
    m{dir}-{amt}
        dir=direction
        t=top
        e/r=end/right
        b=bottom
        s/l=start/left
        x = each side horizontally
        y = each side vertically
      amt=amount (0-5, auto)
  gap (applicable inside d-grid)
    gap-{amt}
      amt=amount (0-5)
